# ჯეტია საქართველო

GTA-style browser driving game set in Tbilisi's Freedom Square.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:8080

## Controls

| Key | Action |
|-----|--------|
| W / ↑ | Accelerate |
| S / ↓ | Brake / Reverse |
| A D / ← → | Steer |
| Space | Handbrake |
| F | Exit / Enter car |
| H | Horn |
| B | Look back |
| C | Cycle camera |
| ESC | Pause |
